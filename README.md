# ☀️ Solar Power Generation Analysis

## Overview

This project analyzes solar power generation using historical plant generation and weather sensor data. The objective is to clean, merge, and analyze the data to identify factors affecting power generation.

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## Dataset

- Plant Generation Data
- Weather Sensor Data
- Cleaned Merged Dataset

## Workflow

- Data Cleaning
- Data Preprocessing
- Data Merging
- Exploratory Data Analysis
- Data Visualization
- Insights

## Project Files

- Solar_Power_Generation_Analysis.ipynb
- Plant_2_Generation_Data.csv
- Plant_2_Weather_Sensor_Data.csv
- cleaned_merged_solar_data.xls
