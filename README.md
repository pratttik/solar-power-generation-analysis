# Solar Power Generation Analysis

## Overview

This project presents a comprehensive analysis of solar power generation using historical plant generation and weather sensor data. The objective is to preprocess, merge, and analyze operational and environmental datasets to identify the key factors influencing solar power generation and provide data-driven insights.

---

## Objectives

- Analyze historical solar power generation data.
- Integrate weather sensor and generation datasets.
- Perform data cleaning and preprocessing.
- Explore relationships between environmental variables and power output.
- Generate visualizations to support data-driven decision-making.

---
